# portfolio_hajar_baraket
# 🌟 HAJAR's World - Portfolio Interactif

Un portfolio créatif sous forme de jeu vidéo. Explore 5 zones (quiz, memory, rythme, whack-a-mole, boss) pour découvrir l'univers de Hajar et débloquer un formulaire de contact et des citations inspirantes.

## 🚀 Fonctionnalités
- Navigation dynamique entre les zones (mini-jeux).
- Quiz de personnalité, memory, rythme, whack-a-mole.
- Boss fight pour débloquer le formulaire de contact.
- Formulaire de contact asynchrone (simulation d'envoi).
- Zone Inspiration : citations récupérées via API (adviceslip, quotable).
- Gestion des états de chargement et d'erreur.
- Responsive design (desktop & mobile).

## 🛠 Technologies utilisées
- HTML5 sémantique
- CSS3 (Flexbox, Grid, animations)
- JavaScript Vanilla (ES6+)
- API fetch (Advice Slip, Quotable, DummyJSON)
- Git & GitHub Pages
